What I've Built

Serverless Cloud-Native Data Pipeline

Mar 2025 – May 2025

Python Terraform AWS Lambda S3 Redshift CloudWatch
Problem

Manual data ingestion processes are brittle — hard to validate, harder to audit, and impossible to scale without touching the code.

Architecture

Fully serverless ETL pipeline using AWS Transfer Family for ingestion, Lambda for transformation, config-driven validation logic, and Redshift for warehousing — all provisioned end-to-end with Terraform.

Outcome

Automated the full ingestion-to-warehouse workflow with repeatable, auditable validation — zero manual intervention required.

Key Learning

Separating config from logic makes pipelines adaptable to new data sources without code changes — dramatically reducing maintenance overhead.

Real-Time Stock Streaming Dashboard

Feb 2025 – Mar 2025

Python WebSocket Alpaca API InfluxDB AWS S3 Grafana
Problem

Stock data is only valuable at the moment it's generated — batch reports are already stale by the time analysts read them.

Architecture

WebSocket-based streaming pipeline pulling live market data from the Alpaca API, writing to InfluxDB for time-series storage, and surfacing it in a push-based Grafana dashboard with real-time updates.

Outcome

Sub-second data freshness on a live market dashboard — demonstrating production-grade streaming architecture end to end.

Key Learning

InfluxDB's time-series model significantly outperformed relational alternatives for high-frequency tick data — the right database choice matters as much as the pipeline design.

Real-Time Stock Analysis with Kafka

Aug 2024 – Sep 2024

Python Apache Kafka AWS S3 AWS Glue AWS Athena
Problem

Processing high-velocity market events requires a system that won't drop data under load or require full rebuilds to scale throughput.

Architecture

Kafka-based event streaming pipeline ingesting live stock data, backed by AWS S3 for durability, Glue for schema cataloguing, and Athena for ad-hoc SQL analysis across the full dataset.

Outcome

Fault-tolerant, scalable stream processing with a full queryable data layer — analytical workloads run without touching the ingestion layer.

Key Learning

Kafka's consumer group model makes horizontal scaling straightforward — adding consumers scales throughput without any changes to the producer side.

Credit Risk Analysis — ML Classification

Jul 2023 – Aug 2023

Python Scikit-Learn Pandas Matplotlib
Problem

Architecture

Classification pipeline using Scikit-Learn, training and tuning multiple ML models on credit datasets, with Matplotlib visualisations to interpret feature importance and model behaviour.

Outcome
