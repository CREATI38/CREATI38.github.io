# Project: Derrick Portfolio Website

## Goal
Build a professional personal portfolio website for Derrick, targeting data engineering, solution engineer, and solution architect roles.

## Audience
Recruiters, hiring managers, and technical interviewers.

## Deployment
This site should be compatible with GitHub Pages.
Use a simple static site setup with HTML, CSS, and light JavaScript only.

## Design Style
- Clean
- Professional
- Modern
- Technical but approachable
- Recruiter-friendly
- Minimal color palette
- Strong readability

## Core Sections
- Hero / Intro
- About
- Skills
- Projects
- Experience
- Certifications
- Contact

## Portfolio Positioning
Highlight Derrick as:
- Data Engineer
- Solution Engineer / Architect candidate
- Strong in data pipelines, cloud platforms, system design, and translating business needs into technical solutions

## Writing Style
- Concise
- Results-oriented
- Professional
- Avoid buzzword overload
- Prefer problem → solution → impact framing

## Project Card Format
For each project, include:
- Problem
- Architecture / Approach
- Tools / Stack
- Outcome / Impact
- Key learning

## Technical Preferences
- Keep code simple and easy to maintain
- No unnecessary frameworks
- Keep GitHub Pages deployment easy
- Prioritize responsive design
